# Physics-Informed Data Augmentation for Surrogate Modelling
**Uncertainty Quantification in FE Analyses**

This repository implements a physics-informed data augmentation pipeline for nonlinear finite element (FE) response data of a reinforced concrete propped cantilever beam. The augmented dataset is intended for surrogate modelling, uncertainty propagation, and reliability analysis.

---

## 1. Purpose and Motivation

Nonlinear FE simulations of reinforced concrete structures are computationally expensive. This script augments an existing FE dataset by generating physically admissible synthetic samples that preserve force–displacement behaviour, damage evolution, and material consistency.

---

## 2. Input Data Structure

### Input parameters
processed_inputs_4.csv:
- sample_id
- fc (MPa)
- c_nom_top_mm (mm)
- c_nom_bottom_mm (mm)

Elastic modulus E is recomputed from fc using the Eurocode relation.

---

## 3. Physical Consistency

Elastic modulus:
E = 22000 (fc / 10)^0.3

Curves are reparameterized using arc-length to handle snap-back safely.

---

## 4. Augmentation Method

Augmented responses are generated by convex interpolation:
y_aug = y_a + alpha (y_b - y_a), alpha in [alpha_min, alpha_max]

Input parameters are mixed consistently, and dependent quantities are recomputed.
